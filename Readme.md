# Azure AI Foundry VM Agent

An AI infrastructure agent built using:

- Azure AI Foundry
- GPT-5 Mini
- FastMCP
- Model Context Protocol (MCP)
- ngrok
- Python

## Current Features

- Remote MCP Server
- hello() tool
- Azure AI Foundry integration
- Human approval before tool execution
